# AI Marketing Agent — Skill Reference

Full-stack AI marketing toolkit — scout X/Twitter and Reddit for trending topics, discover and deep-analyze competitors, find content gaps, publish SEO- and GEO-optimized articles with AI illustrations and voice-over in 55 languages, create social media adaptations, generate lead magnets, ingest any URL into structured content, generate short-form AI avatar videos with subtitles, and run fully automated content autopilot.

**Version**: 3.0.0 | **Author**: Citedy

## Quick Start

### 1. Register

```bash
curl -X POST https://www.citedy.com/api/agent/register \
  -H "Content-Type: application/json" \
  -d '{"email": "you@example.com", "blogHandle": "myblog"}'
```

### 2. Configure

Set your API key as `CITEDY_API_KEY` in your agent environment, then visit the [dashboard](https://www.citedy.com/dashboard/settings) to:
- Connect social accounts (X, LinkedIn, Facebook, Instagram, YouTube, Reddit, Threads)
- Upload product knowledge documents
- Set content preferences (tone, language, image style)

### 3. Start Creating

Use the [command reference](/platform/overview#command-reference) or [MCP tools](/mcp/tools) to create content.
