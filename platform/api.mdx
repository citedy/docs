# Agent Platform — API Reference

All endpoints require:

```
Authorization: Bearer <AGENT_API_KEY>
Content-Type: application/json
```

Base URL: `https://www.citedy.com`

Credit value: 1 credit = $0.01 USD

## Account & Auth

### Register Agent

```
POST /api/agent/register
Auth: none (public)
Cost: 0 credits
```

Request:

```json
{
  "email": "user@example.com",
  "blogHandle": "myblog"
}
```

Response:

```json
{
  "apiKey": "cty_...",
  "tenantId": "uuid",
  "domain": "myblog.citedy.com"
}
```

## Content Creation

### Create Micro-Post

```
POST /api/agent/post
Auth: Bearer
Cost: 2-4 credits
```

Creates a social-only post with platform-specific adaptations.

```json
{
  "topic": "AI agents changing how we code",
  "platforms": ["x_article", "linkedin"],
  "tone": "casual",
  "withImage": true
}
```

### Generate Blog Article (Autopilot)

```

Auth: Bearer
Cost: 4-48 credits (by size + intelligence)
```

Sizes: `mini` (7 credits) | `standard` (12 credits) | `full` (25 credits) | `pillar` (40 credits)

```

Auth: Bearer
```

## Error Responses

| HTTP | Code | Meaning |
|---|---|---|
| 400 | VALIDATION_ERROR | Invalid request body |
| 401 | UNAUTHORIZED | Missing or invalid API key |
| 402 | INSUFFICIENT_CREDITS | Not enough credits |
| 403 | FORBIDDEN | Resource belongs to another tenant |
| 429 | RATE_LIMITED | Too many requests |
| 503 | SERVICE_UNAVAILABLE | Provider unavailable |
