# Citedy Agent Platform

**Version**: 1.7.0 | **Status**: Production

Citedy is a backend service for AI agents. Connect once, configure defaults in the web UI, and let any agent — Claude Code, Cursor, ChatGPT, or any MCP-compatible client — write posts, generate images, publish to social platforms, scan trends, and produce short-form video.

The web UI is for settings and scheduling review. The agent does the work.

## Authentication

All agent endpoints require:

```
Authorization: Bearer <AGENT_API_KEY>
```

Obtain your key at [citedy.com/dashboard/settings](https://www.citedy.com/dashboard/settings) or via `POST /api/agent/register`.

Key rotation: `POST /api/agent/rotate-key`
