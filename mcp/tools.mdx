title: MCP Tools Reference
description: Complete list of 38 MCP tools available via the Citedy MCP server
---


The Citedy MCP server exposes 38 tools. Each tool maps to one or more Agent API endpoints.

## Content Creation

| Tool | Description | API Endpoint |
|---|---|---|
| `article.generate` | Generate a new article using autopilot | POST /api/agent/autopilot |
| `article.list` | List previously generated articles | GET /api/agent/articles |
| `adapt.generate` | Generate social adaptations for an article | POST /api/agent/adapt |

## Lead Magnets

| Tool | Description | API Endpoint |
|---|---|---|
| `leadmagnet.generate` | Start lead magnet generation (checklist/swipe/framework) | POST /api/agent/lead-magnets |
| `leadmagnet.get` | Fetch lead magnet status/result by ID | GET /api/agent/lead-magnets/[id] |
| `leadmagnet.publish` | Publish a lead magnet | PATCH /api/agent/lead-magnets/[id] |
| `leadmagnet.archive` | Archive a lead magnet | PATCH /api/agent/lead-magnets/[id] |

## Async Operations

Some tools return `"status": "processing"` with polling instructions:

- `shorts.generate` → poll with `shorts.get`
- `leadmagnet.generate` → poll with `leadmagnet.get`
- `scout.x` → poll with `scout.x.result`
- `scout.reddit` → poll with `scout.reddit.result`
- `ingest.create` → poll with `ingest.get`
