---

## 1. Register

```bash
curl -X POST https://www.citedy.com/api/agent/register \
  -H "Content-Type: application/json" \
  -d '{"agent_name": "my-agent"}'
```

Response:

```json
{
  "pending_id": "...",
  "approval_url": "https://www.citedy.com/approve-agent/...",
  "expires_in": 3600
}
```

Open `approval_url` in browser, approve the agent, and copy the API key shown on the page.

## 2. Check Health

```bash
curl https://www.citedy.com/api/agent/health \
  -H "Authorization: Bearer YOUR_API_KEY"
```

## 3. Check Operational Status

```bash
curl https://www.citedy.com/api/agent/status \
  -H "Authorization: Bearer YOUR_API_KEY"
```

Returns an actionable snapshot: credit balance, connected social accounts, schedule gaps, and recommended next actions.

## 4. Create Your First Post

```bash
curl -X POST https://www.citedy.com/api/agent/post \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "topic": "AI agents changing how we code",
    "platforms": ["x_article", "linkedin"],
    "tone": "casual"
  }'
```

## 5. Publish

```bash
curl -X POST https://www.citedy.com/api/agent/publish \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "adaptationId": "ADAPTATION_ID_FROM_STEP_4",
    "action": "now",
    "platform": "linkedin",
    "accountId": "YOUR_ACCOUNT_UUID"
  }'
```
