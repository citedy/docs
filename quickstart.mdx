## 1. Create Account & Get API Key

1. Sign up at [citedy.com](https://www.citedy.com)
2. Go to [Dashboard → Settings → Agent](https://www.citedy.com/dashboard/settings?section=team&agent=connect)
3. Connect an agent and copy your API key (`citedy_agent_...`)

New accounts receive **100 free credits** — no card required.

## 2. Check Operational Status

```bash
curl https://www.citedy.com/api/agent/status \
  -H "Authorization: Bearer YOUR_API_KEY"
```

Returns credit balance, connected social accounts, schedule gaps, and recommended next actions.

## 3. Upload Product Knowledge (optional, 1 credit)

```bash
curl -X POST https://www.citedy.com/api/agent/products \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{"url": "https://docs.yourproduct.com"}'
```

## 4. Write Your First Article (~20 credits)

```bash
curl -X POST https://www.citedy.com/api/agent/autopilot \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{"topic": "AI agents changing how we code", "size": "standard"}'
```

## 5. Adapt for Social (~5 credits/platform)

```bash
curl -X POST https://www.citedy.com/api/agent/adapt \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{"article_id": "ARTICLE_ID_FROM_STEP_4", "platforms": ["linkedin", "x_article"]}'
```

## 6. Publish (free)

```bash
curl -X POST https://www.citedy.com/api/agent/publish \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{"adaptationId": "ADAPTATION_ID", "action": "now", "platform": "linkedin"}'
```

Total cost: ~32 credits = **$0.32** for an SEO article published with social posts.
