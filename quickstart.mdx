# Quick Start

## 1. Register

```bash
curl -X POST https://www.citedy.com/api/agent/register \
  -H "Content-Type: application/json" \
  -d '{"email": "you@example.com", "blogHandle": "myblog"}'
```

Response:

```json
{
  "apiKey": "cty_...",
  "tenantId": "uuid",
  "domain": "myblog.citedy.com"
}
```

Save your `apiKey` — you'll need it for all subsequent requests.

## 2. Check Health

```bash
curl https://www.citedy.com/api/agent/health \
  -H "Authorization: Bearer YOUR_API_KEY"
```

## 3. Check Operational Status

```bash
curl https://www.citedy.com/api/agent/status \
  -H "Authorization: Bearer YOUR_API_KEY"
```

This returns an actionable snapshot: credit balance, connected social accounts, schedule gaps, and recommended next actions.

## 4. Create Your First Post

```bash
curl -X POST https://www.citedy.com/api/agent/post \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "topic": "AI agents changing how we code",
    "platforms": ["x_article", "linkedin"],
    "tone": "casual",
    "withImage": true
  }'
```

## 5. Publish

```bash
curl -X POST https://www.citedy.com/api/agent/publish \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "contentId": "POST_ID_FROM_STEP_4",
    "action": "now",
    "platforms": ["x_article", "linkedin"]
  }'
```
