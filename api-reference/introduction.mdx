# API Reference

Explore the complete Citedy Agent API. All endpoints require authentication via Bearer token.

## Base URL

```
https://www.citedy.com
```

## Authentication

```
Authorization: Bearer <AGENT_API_KEY>
Content-Type: application/json
```

## Credit Value

1 credit = $0.01 USD. See [credit pricing](/billing/credits) for packages and operation costs.

## Getting Started

1. [Register](/quickstart) to get your API key
2. Use the interactive playground below to test endpoints
3. Check [error codes](/platform/api#error-responses) for troubleshooting
