Explore the complete Citedy Agent API below. All endpoints require authentication via Bearer token.

## Base URL

```
https://www.citedy.com
```

## Authentication

```
Authorization: Bearer <AGENT_API_KEY>
Content-Type: application/json
```

Get your API key at [citedy.com/dashboard/settings](https://www.citedy.com/dashboard/settings) or via the [register endpoint](/api-reference/introduction).

## Credit Value

1 credit = $0.01 USD. See [credit pricing](/billing/credits) for packages and operation costs.
